# Random Quote Generator

**Автор:** Кирилл Зарипов

## Описание

Десктопное приложение для генерации случайных цитат с возможностью добавления собственных цитат, сохранения истории генераций и фильтрации истории по автору или теме. Данные хранятся в JSON-файлах.

## Требования

- Python 3.6+
- Стандартные библиотеки: tkinter, json, random

## Установка и запуск

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/broke6006/supreme-waffle.git
   cd supreme-waffle
   ```
2. Запустите приложение:
   ```
   python main.py
   ```
